# EDA-for-Google-Ecommerce-Site
This project focuses on exploratory data analysis of Google Ecommerce dataset to understand user behavior and extract business insights.

## 🔧 Data Cleaning
* Handled missing values by filling nulls with 0
* Removed duplicate records
* Ensured consistent data types and structure

## 📈 Exploratory Data Analysis
* **Numeric Analysis:** Studied distributions and trends in numerical features
* **Categorical Analysis:** Analyzed user behavior across different categories (device, traffic source, etc.)

## 🧠 Feature Engineering
* Created **Engagement Level** feature
* Calculated **Hits per Page** to measure user interaction

## 🤖 Predictive Modeling
* Built a Logistic Regression model to predict user conversion

## 💡 Key Insights & Recommendations
* Desktop users show high engagement
* Chrome users have the highest engagement and Opera show the lowest among all the browser.
* Pageview and hits are highly correlated
